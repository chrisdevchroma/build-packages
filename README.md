# Build packages with containers

## Dockerfiles

Set of containers with Development Tools packages installed
* CentOS 8 (8.1 1911) 
* CentOS 7 (7.7 1908)
* Fedora 31 

These containers can be used to build rpm's of:
* Deluge
* Discord

And their dependencies which are not in rpm-fusion or EPEL
* rb_libtorrent
* asio
* libcxx
* libcxxabi
